# CS110 A0/B1 Final Exam

## DESCRIPTION
This program is meant to help you find a new book based off of a subject and a new movie to go along with your book. Using two API's the program presents the user an interactive TUI experience.

## KNOWN BUGS AND INCOMPLETE PARTS
Most functionality is contained within the main driver code to keep variables in the global scope.

## REFERENCES

## MISCELLANEOUS COMMENTS
If the API is down and is not able to return content to the program, the program will purposefully push an exception.
